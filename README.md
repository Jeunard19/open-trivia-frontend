This project is a **React-based frontend application** for the Open Trivia API backend.  
It provides a simple user interface that allows users to fetch trivia questions and submit answers for validation.

The application is fully **Dockerized** and runs using **Docker Compose**.

---

##  Project Description


The UI allows users to:

- Request a trivia question
- View multiple-choice options
- Select or enter an answer
- Submit the answer and receive feedback

---

## Running the Application with Docker

### Build & Run the Frontend

```bash
docker-compose up --build
```
